# UG LAB

## 起動方法

1. このフォルダをVS Codeで開く
2. `index.html` を右クリック
3. `Open with Live Server`

## 構成

```text
index.html
css/style.css
data/solarTerms.js
data/ugdb.js
js/calendar.js
js/bodyos.js
js/reports.js
js/app.js
assets/
```

## 方針

- スタークローラー比較なし
- 境界テストなし
- デバッグ表示なし
- 患者レポートに命式なし
- セルフケアと次回確認はUG判定から自動提案

答えは身体にある。
