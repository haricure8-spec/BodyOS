# UGLAB USB 3588 再構築版

## 起動
1. ZIPを右クリックして「すべて展開」します。
2. 展開した `UGLAB-USB-3588-Rebuilt` フォルダを丸ごとUSBへコピーします。
3. USB内の `START-UGLAB.cmd` をダブルクリックします。
4. 黒い画面は使用中ずっと閉じないでください。

院内画面は `http://localhost:3588/` です。
受付QRは院内画面に表示されるものを使用します。

## 終了
`STOP-UGLAB.cmd` をダブルクリックしてからUSBを安全に取り外してください。

## 保存場所
患者・受付・施術記録はすべてUSB内の `UGLAB_DATA` に保存されます。
このフォルダは削除しないでください。

## 重要
- 3000番ポートは使いません。
- 起動ファイルは日本語コマンドとPowerShellを使っていません。
- ZIPの中から直接起動せず、必ず「すべて展開」してから起動してください。
